## Key Features

### Hero Section
- **Headline:** "Turn Local Businesses Into $5K Paychecks"
- **Subheadline:** Clear value proposition explaining the business model
- **Primary CTA:** "Start Free Audit" (gold gradient button)
- **Secondary CTA:** "See Demo Dashboard"
- **Social Proof:** 3 trust signals (3-day close time, 87% response rate, Stripe secure)

### ROI Calculator
- Interactive slider (5-50 leads per week)
- Real-time calculations showing:
  - Leads Found
  - Deals Closed (30% conversion rate)
  - Monthly Revenue potential
- Helps visitors visualize earning potential

### How It Works (4 Steps)
1. **Find Targets** - Google Maps scraper
2. **AI Audits** - Automated website analysis
3. **Send Invoice** - One-click payment links
4. **Get Paid** - Automatic webhook updates

### Pricing Section
Three tiers with clear feature comparison:
- **Basic:** $3,000 (single-page website)
- **Standard:** $5,000 (MOST POPULAR badge)
- **Premium:** $8,000 (BEST VALUE badge)

Each tier shows:
- Price
- Description
- Feature checklist with checkmarks
- Delivery timeline
- Support duration

### Final CTA
- Large card with gradient background
- "Ready to Start Making Money?" headline
- "Get Dashboard Access" button
- Urgency text: "First audit is FREE • No credit card required"
