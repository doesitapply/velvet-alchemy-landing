## 🎥 RECORDING PROCESS

### **Step 1: Set Up Loom**
1. Install Loom browser extension
2. Click Loom icon in toolbar
3. Select "Screen Only" recording
4. Choose "Full Screen" or "Current Tab"
5. Test microphone (speak and check levels)

### **Step 2: Start Recording**
1. Click "Start Recording" in Loom
2. Wait for 3-second countdown
3. Begin narration immediately
4. Follow script and shot list exactly

### **Step 3: Navigate Interface**
- **Move mouse slowly** (easier to follow)
- **Pause briefly** before clicking (build anticipation)
- **Highlight important elements** (hover over them)
- **Don't rush** (90 seconds is plenty of time)

### **Step 4: End Recording**
1. Finish narration at 1:30
2. Hold on final frame for 2 seconds
3. Click "Stop Recording" in Loom
4. Loom automatically processes and uploads

### **Step 5: Review & Edit**
1. Watch full video in Loom
2. Use Loom's trim tool to cut mistakes
3. Add title: "Velvet Alchemy Demo - $5K in 3 Days"
4. Add description: "Watch how to turn local businesses into $5,000 paychecks using AI-powered website audits and automated invoicing."

## 🎬 ADVANCED TIPS

### **Visual Polish:**
- **Cursor Highlighting:** Use a cursor highlighter tool (like Mouseposé) to make clicks more visible
- **Zoom Effects:** Use Loom's zoom feature to highlight important UI elements
- **Smooth Scrolling:** Scroll slowly and steadily (not jumpy)
- **Clean Transitions:** Pause 0.5 seconds between major scene changes

### **Audio Quality:**
- **Microphone:** Use external mic if possible (Blue Yeti, Rode NT-USB)
- **Environment:** Record in quiet room (no background noise)
- **Mouth Distance:** 6-8 inches from mic (not too close)
- **Pop Filter:** Use one to reduce "p" and "b" sounds

### **Narration Delivery:**
- **Smile while talking** (makes voice sound friendlier)
- **Stand up while recording** (better breath control)
- **Read from script on second monitor** (don't memorize)
- **Take breaks** (record in segments if needed)
